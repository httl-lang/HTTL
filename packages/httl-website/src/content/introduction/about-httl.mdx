# What is HTTL?

HTTL ("HyperText Transfer Language") is a lightweight, programmng language designed to for developers who prefers using code to operate with API rathen then UI.\
\
HTTL is a declarative language that allows you to write queries to the API in a way that is similar to SQL.\
The same as in SQL, HTTL allows you to write HTTP queries, set the result in variables and use them in the next queries.\
\
HTTL also can be use as integration language between different systems, as it allows to write queries to different APIs in one file.
